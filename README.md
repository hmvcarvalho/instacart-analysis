# Instacart Market Basket Analysis

## Project Description
This project analyzes customer shopping behavior using Instacart's publicly released dataset. The analysis focuses on data cleaning, exploratory data analysis (EDA), and generating insights about customer purchasing patterns on the grocery delivery platform.

The dataset was originally released by Instacart in 2017 for a Kaggle competition and contains information about customer orders, products, and shopping habits.

## Data Files Used

- **instacart_orders.csv** - Contains order information including order timing and customer details
- **products.csv** - Product catalog with product names and specifications  
- **order_products.csv** - Links between orders and products, showing what was purchased
- **aisles.csv** - Store aisle information for product categorization
- **departments.csv** - Department classification for products

## Project Goals
- Clean and preprocess the data
- Analyze customer shopping patterns
- Generate insights about product preferences and ordering behavior
- Create visualizations to present findings
